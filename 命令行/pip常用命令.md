# pip安装
* [参考](https://pip.pypa.io/en/stable/installing/) 
* 运行命令,下载get-pip.py
```
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
```
* 安装pip 
```
py get-pip.py 
```
* 配置环境变量\python\Scripts 
* 检查pip版本
```
$ pip --version
pip 21.1.3 from d:\mysoftware\python\lib\site-packages\pip (python 3.6)
```
