# list和str相互转换

```
a = ['1', '2', '3', '', '5']
b = ','.join(a)
print(b, type(b), len(b))
c = b.split(',')
print(c, type(c), len(c))

d = ['6']
e = ','.join(d)
print(e)
```
```
>>1,2,3,,5 <class 'str'> 8
>>['1', '2', '3', '', '5'] <class 'list'> 5
>>6
```

# 简单计算程序运行时间

```
import time
import datetime

star = datetime.datetime.now()
time.sleep(2)
end = datetime.datetime.now()
print(star)
print(end)
print(end - star)
```
```
2020-08-03 19:27:57.865100
2020-08-03 19:27:59.865903
0:00:02.000803
```
