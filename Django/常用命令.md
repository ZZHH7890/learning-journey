# 常用命令

**安装**

```
pip install django
```
**查看版本号**
```
 pip show Django
```
**创建项目**

```
django-admin startproject guest
```
**创建应用,需要进入guest项目目录**

```
django-admin startapp sign
```

**启动服务器**
```
$ py manage.py runserver
```
浏览器输入 http://127.0.0.1:8000/
